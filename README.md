# Coffee Shop Requirements

This repository currently contains the collected requirements and starter branding for a coffee shop concept.

- Requirements brief: `branding/requirements.md`
- Logo asset: `branding/logo.svg`
- Static site entry point: `index.html`

## Deployment

The site is deployed automatically with GitHub Actions to GitHub Pages on every push to the `main` branch.

After the first successful workflow run, the published site will be available from the repository's GitHub Pages URL.
